    Represents scan results from the API.

    Constructor

    new ScanResults(data)

    Creates scan results

    Parameters:
    Name Type Description
    data Response

    The response from the API when performing GET /files/{id}. Warning: The Response MUST have been resolved!
    Properties
    Name Type Description
    results Object

    The raw scan results
    stats Object

    The last analysis stats

    Properties
    Name Type Description
    harmless Number

    The number of engines that say OK.
    malicious Number

    The number of engines that flag the file.
    suspicious Number

    The number of engines that are suspicious about the file.
    timeout Number

    The number of engined that timed out.
    unsupported Number

    The number of engines that don't support that type.
    undetected Number

    The engines that say this file is OK.
